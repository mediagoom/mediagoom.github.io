How it works

{{ page.description }}
It's composed of several projects. The main ones are:
node-play
node-play is a Streaming Server.
It allows to upload a video file and have it converted and streamed on the web.
mg
mg is a media packager.
It allows to package a series of mp4 files into hls or mpeg dash.

Getting Started

The simplest way to get started is to install node-play. In order to install node-play please follow the instruction you can find here.
